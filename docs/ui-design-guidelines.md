# UI & Design Guidelines

Germany Move Quest now follows a consistent visual language.

## Icons
Use Lucide icons throughout the application.
Do not import icons directly into presentation components unless there is a compelling reason.
Centralize icon choices in the Icon Catalog.
Reuse icons consistently across the application.
Prefer meaningful symbolic icons over literal decorative icons.

## Current icon language includes:

Navigation
Journey stages
Completion indicators

Future Profile, Home Setup and Inventory features should also use the Icon Catalog.

## Bilingual Design

The application is German-first.

Where appropriate:

Use the German term for concepts the user will encounter in Germany. 
Use English for guidance, explanations, and instructional text. 
The UI should naturally help users learn everyday German terminology during their relocation.

Page titles should match the navigation labels.

## Visual Style

Continue favouring:

calm, uncluttered layouts
generous whitespace
restrained colour usage
icons that communicate meaning rather than decoration
reusable cards
consistent spacing and typography
mobile-first responsive layouts

Avoid unnecessary visual complexity.

## Journey Philosophy

The journey does not end after the move.

The Journey Progress timeline intentionally extends beyond the final stage with an arrow to reinforce that Germany Move Quest is a relocation companion, not simply a relocation checklist.
